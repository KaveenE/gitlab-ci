# Gitlab CI/CD
A 0 to production tutorial using Gitlab for management of continuous integration and continuous deployment. In this tutorial our setup enviroment will be:
- GitLab `(Source base code)`
- Digital Ocean `(VM host)`
- Docker `(Containers)`
